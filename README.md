### Desafio - GFT Start Woman Java

**Descrição do desafio**
Agora é a sua hora de brilhar! Crie uma solução que explore o conceito de Padrões de Projeto na prática. Para isso, você pode reproduzir um dos projetos que criamos durante as aulas ou, caso se sinta preparado, desenvolver uma nova ideia do zero.

## Projeto - Condomínios Api
Essa é uma API integrada com o banco de dados MySQL que visa integrar dados de condomínios, como idCondominio, cnpj, razão social e cep. São realizados os serviços de cadastrar, atualizar, consultar todos os condomínios e consultar por um identificador, e deletar.

##### Objetivos futuros
Realizar operações com outras entidades e interfaces. E projetar o front-end com Angular.
